Syntax

Code Syntax

  • Each instruction will be interpreted line by line
  • The operation and operand are seperated by space
  • Operations are case-insensitive
  • Operands are case sensitive

Register Syntax

  • Register and its value are seperated by space
  • Register value can be null
  • Example:
    A
    B 10
    C A+B

Modes

  • Two types of modes are there:
    • Numeric mode
    • Non-numeric mode
  • Numeric mode applies operation on the values.
    Example:
    • Register file
      A 10
      B 20
      C
    • Code file
      push A
      push B
      add
      pop C
    • After code execution C = 30
  • Non-numeric mode applies operation on the register variables.
    Example:
    • Register file
      A 10
      B 20
      C
    • Code file
      push A
      push B
      add
      pop C
    • After code execution C = B+A

General

  • Always run the register file after changing content to update the changes
  • Executng a code will modify the register file
